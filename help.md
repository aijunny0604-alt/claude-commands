# 커스텀 스킬 도움말

현재 사용 가능한 커스텀 스킬 목록을 보여주세요.

## 커스텀 스킬 목록

| 명령어 | 용도 | PDCA | 설명 |
|--------|------|:----:|------|
| `/full-test` | 대규모 통합 테스트 | ✅ | 4개 에이전트 팀 로컬+프로덕션 API 테스트 |
| `/security-team` | 보안 팀 점검 | ✅ | 3개 에이전트 팀 OWASP Top 10 풀 스캔 |
| `/security-quick` | 빠른 보안 | ✅ | 1인 경량 보안 점검 (5분) |
| `/mobile-audit` | 모바일 최적화 | ✅ | 4개 에이전트 팀 모바일 UI/UX |
| `/responsive-check` | 반응형 점검 | ✅ | 3개 해상도 자동 스크린샷 |
| `/pre-deploy` | 배포 전 체크 | ✅ | 빌드+타입+DB+환경변수 자동 검증 |
| `/quick-fix` | 빠른 버그 수정 | ✅ | 원인 추적→수정→검증 자동화 |
| `/check-pos` | POS 점검 | - | POS Calculator 앱 전용 |
| `/help` | 도움말 | - | 이 목록 표시 |

### PDCA 사이클이란?
Plan(계획) → Do(실행) → Check(분석) → Act(수정) → 90점 이상까지 자동 반복

위 목록을 보기 좋게 표시하세요.
