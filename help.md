### v2.1 공통 강화 사항 (전 에이전트 적용)

1. **영향도 맵 (Impact Map)**: 점검 전 코드 연쇄 관계를 시각화하여 누락 방지
2. **시나리오 매트릭스**: 변수 축(X/Y/Z) 조합으로 테스트 시나리오 자동 도출
3. **마스터 체크리스트**: 시나리오에서 실행 가능한 체크 항목 자동 생성 + 커버리지 추적
4. **교차 검증**: 팀 간 결과를 교차 대조하여 모순/누락 탐지
5. **영향도 역추적**
6. **Playwright UI E2E** — 실제 브라우저에서 클릭/입력/모달/데이터 검증 (playwright-cli 스킬): FAIL 발견 시 영향도 맵 역방향 추적으로 파급 범위 확인

### PDCA 사이클이란?
Plan(계획) → Do(실행) → Check(분석) → Act(수정) → 90점 이상까지 자동 반복
