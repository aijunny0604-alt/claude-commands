# 배포 전 자동 체크리스트 - PDCA 배포 게이트

배포 전 필수 점검을 자동으로 수행합니다. 모든 항목 PASS여야 배포를 허용합니다.

인자: $ARGUMENTS ("local" 또는 "production")

---

## 체크리스트 자동 실행

### 1. 빌드 체크
```bash
npx next build
```
❌ 실패 시 → 에러 수정 → 재빌드 (최대 3회)

### 2. 타입 체크
```bash
npx tsc --noEmit
```
타입 에러 0건 확인.

### 3. 미커밋 변경사항 확인
```bash
git status
git diff --stat
```
커밋 안 된 파일이 있으면 경고.

### 4. 환경변수 확인
- .env.local 존재 여부
- 필수 환경변수 누락 여부 (CLAUDE.md 기준)

### 5. DB 스키마 동기화
```bash
npx prisma validate
```
스키마 에러 없는지 확인.

### 6. 최근 커밋 리뷰
```bash
git log --oneline -5
git diff HEAD~1 --stat
```
변경된 파일 목록 + 위험도 평가.

### 7. 프로덕션 헬스체크 (production일 때만)
```bash
curl -s -o /dev/null -w "%{http_code}" {프로덕션URL}
```
200 응답 확인.
